# @qiwi/tech-radar

Tech radar generator

## Usage

```
yarn add @qiwi/tech-radar
techradar  --input "/path/to/files/*.{json, csv, yml}" --output /radar
```
or
```
npx @qiwi/tech-radar --input "/path/to/files/*.{json, csv, yml}" --output /radar
```

| Args | description |
|---|---
| input | path to csv/json/yml file
| output |
